DESCRIÇÃO
Neste Projeto de LAB você será desafiado a construir um sistema para um estacionamento, 
que será usado para gerenciar os veículos estacionados e realizar suas operações, 
como por exemplo adicionar um veículo, remover um veículo (e exibir o valor cobrado durante o período) e listar os veículos.

…or create a new repository on the command line
echo "# DIO-Construindo-um-Sistema-para-um-Estacionamento-com-C-" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/prof-pccarvalho/DIO-Construindo-um-Sistema-para-um-Estacionamento-com-C-.git
git push -u origin main


…or push an existing repository from the command line
git remote add origin https://github.com/prof-pccarvalho/DIO-Construindo-um-Sistema-para-um-Estacionamento-com-C-.git
git branch -M main
git push -u origin main
